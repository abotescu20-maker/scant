# Expressionism (Die Brücke and Der Blaue Reiter)

**Period:** 1905-1920s

## Style

- German response — Van Gogh plus Munch plus African and medieval wood sculpture
- Die Brücke (Dresden 1905): harsh, angular, urban angst
- Der Blaue Reiter (Munich 1911): spiritual, lyrical, tending toward abstraction
- art must express inner necessity not outer appearance

## Key artists

- Kirchner
- Nolde
- Schmidt-Rottluff
- Heckel (Die Brücke)
- [[kandinsky|Kandinsky]]
- Franz Marc
- [[klee|Klee]]
- Macke (Der Blaue Reiter)
- Kokoschka
- Beckmann

## Techniques

- jagged angular drawing and deliberate 'crudeness' of form
- strident clashing colour — acid greens, magentas, cold blues
- woodcut with coarse tool marks left visible as expressive element
- distorted faces, elongated limbs, primitive mask-like features
- Kandinsky's spiritual abstraction: untethered colour-shapes with musical titles ('Composition', 'Improvisation')
- Marc's animals in symbolic colour (blue horse, red deer)
