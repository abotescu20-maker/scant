# Van Gogh / Expressive Post-Impressionism

**Period:** 1880-1890

## Style

- colour and brushstroke as vehicles of emotion, not description
- each object given its own pulsating rhythm
- paint as direct record of feeling and of the painter's hand

## Key artists

- [[van-gogh|Van Gogh]]

## Techniques

- thick impasto ridges squeezed straight from the tube
- short, rhythmic parallel strokes that swirl in cypresses, stars, skies
- pure saturated colour — chrome yellow, cobalt blue, viridian — with dark contour
- reed-pen drawings of dots, dashes, curls echoing the painted stroke
