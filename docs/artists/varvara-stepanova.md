# Varvara Stepanova

**Dates:** 1894-1958
**Movement:** [[constructivism|Russian Constructivism]] / Productivism

## Signature style

- Textile and clothing design for the First State Textile Factory (1924) — bold geometric repeats
- "Production clothing" (*prozodezhda*) — utilitarian, unisex garments for workers and athletes
- Book design and poetry (*Gaust chaba*, 1919) — abstract lettering as image
- Theatre designs for *The Death of Tarelkin* (1922) — kinetic constructivist set and costume
- Married to and collaborated with [[alexander-rodchenko|Rodchenko]]; co-edited magazine *LEF*
- Flat geometric forms: circles, triangles, stripes — rhythm based on mechanical repetition

## Visual signatures for AI

- High-contrast geometric repeat patterns (on fabric or page)
- Red, black, white, orange palette
- Flat shapes with no modelling or gradient
- Typography set into geometric grids
