# Kandinsky

**Dates:** 
**Movement:** [[expressionism|Der Blaue Reiter Expressionism → Abstract]]

## Signature style

- first fully non-objective watercolour c. 1910-13
- floating colourful shapes: arcs, zig-zags, checkerboards with lyrical titles
- Bauhaus geometric period (1920s): circles, triangles, sharp lines
- wrote 'Concerning the Spiritual in Art'
