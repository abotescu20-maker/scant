# Eugène Delacroix

**Dates:** 1798-1863
**Movement:** [[romanticism|French Romanticism]]

## Signature style

- complementary colour pairs (orange/blue, red/green) juxtaposed to vibrate
- orientalist subjects from Moroccan journey
- violent action — lions, horses, massacres
- rapid broken brushwork — no licked finish
- 'colourist' flag to Ingres' 'linear'
