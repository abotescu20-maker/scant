# Goya

**Dates:** 
**Movement:** [[rococo|late Rococo → Romanticism]]

## Signature style

- black palette of ivory black, ochre, blood red
- loose slashing brushwork anticipating Manet
- nightmare subjects: witches, madmen, firing squads
- court portraits that mercilessly record vanity
- aquatint etchings with dense velvety blacks (Los Caprichos, Disasters of War)
