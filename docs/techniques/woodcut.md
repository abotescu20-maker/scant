# Woodcut (modern Expressionist)

## Technique

Deliberate roughness — splinters, tool marks, coarse grain — left visible as expressive element, in contrast to the smooth refinement of 19th-century wood engraving.

## Practitioners

- [[gauguin|Gauguin]]
- Munch
- Die Brücke (Kirchner, Heckel, Nolde)
