# Lithography

## Technique

Drawing with a greasy crayon on limestone so that ink adheres only to the drawn areas — enabled mass-produced tonal images. Exploited for political satire (Daumier) and for poster design (Toulouse-Lautrec, Mucha).

- Toulouse-Lautrec
