# Registration Misalignment

## Technique

Because each Risograph colour is printed on a separate pass — paper is taken out, the drum swapped, the paper reloaded — the second, third and fourth layers never line up exactly with the first. Typical offsets on a small-format Riso are 0.5–3 mm, varying from copy to copy within the same edition.

In precision commercial printing ("offset litho") this would be a defect. On a Riso it is a feature. Designers exploit it in several ways:
- leave a small gap ("trap") between colours so slight drift still reads clean
- deliberately over-extend one layer so it prints *over* the line art, accepting visible coloured halos around black outlines
- design compositions where drifted layers create a secondary, animated sense of depth — each copy in an edition looks subtly different
- pair this with the poor consistency of ink coverage to produce prints that feel hand-made

Detectable signatures:
- bright coloured fringes on one edge of a line-art shape (cyan fringe on the left, say; red fringe on the right)
- text printed as a single spot colour surrounded by a thin offset halo from an underlying layer
- a colour layer sitting proudly of its intended container, producing "ghost" outlines
