# Grid system

## Technique

An underlying modular structure — rows, columns, baseline — used to organise type, image, and white space on the page. In [[swiss-poster-international-typographic-style|Swiss poster design]] the grid is a mathematical tool: decisions about proportion, alignment, and hierarchy are derived from the grid rather than from "artistic" intuition.

Characteristic features:

- Page divided into a field of uniform columns (often 8 or 12) and rows
- Every element (headline, photograph, body text, caption) aligns to at least one grid line
- Consistent baseline grid harmonises mixed type sizes
- Flush-left, ragged-right paragraph setting
- Generous negative space built into the grid, not added afterward
- Ideally the grid is invisible but perceptible — "rational order"
