# foreshortening

## Technique

Drawing an object or limb receding sharply into depth so that near parts are enlarged and far parts compressed, producing the illusion of projection toward the viewer. First essayed on Greek red-figure vases (a foot seen from above), perfected by Hellenistic, Roman and then Renaissance painters.

## Practitioners

- Euphronios
- Giotto (early)
- Mantegna
