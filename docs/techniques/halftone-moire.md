# Halftone Moiré

## Technique

A Risograph is a binary printer: at any given point, a spot colour either fires through the [[stencil-master|master]] or it does not. To reproduce tones — a 40 % mint wash, a soft pink gradient — the file must be broken into halftone dots: small dots of 100 % ink whose *density*, not whose colour, varies across the image.

Because each ink is screened independently, and because [[registration-misalignment|layers shift slightly between passes]], the overlapping dot grids of different colour layers interact. When two screens sit at similar angles or frequencies, they produce a visible *moiré* pattern — wavy, swirling interference fringes superimposed on the image.

Riso designers handle moiré in two ways:
- rotate each colour's halftone screen to a different angle (traditionally 15°, 45°, 75°) to minimise interference, as offset lithography does
- deliberately leave screens at the same angle and use the moiré as a texture — visible ripples become part of the aesthetic

Typical visual signatures:
- coarse, visible dot grid in anything that should be a smooth tone
- wavy interference fringes across overlapping colour fields
- grain that is *regular* (a printed screen) rather than random (film grain)
- soft edges where a halftone gradient meets a solid
