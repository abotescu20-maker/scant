# figura serpentinata

**Period:** Mannerism (16th c.)

## Technique

compositional rule that the body should twist in a spiralling S-shape from feet to head, showing as many viewpoints as possible in one figure.

## Visual signature

elongated spine curving; shoulders and hips turned in opposing directions; the figure feels like a sculpture designed to be walked around.

## Practitioners

- Giambologna
- Parmigianino
- Michelangelo (precursor)
