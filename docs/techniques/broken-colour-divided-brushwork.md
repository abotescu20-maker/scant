# Broken colour / divided brushwork

## Technique

Small discrete strokes of unmixed pigment laid side by side so that they fuse optically at viewing distance into a luminous vibrating whole. Parent of all modern colour practice.

## Practitioners

- Impressionists
- [[seurat|Seurat]]
- [[van-gogh|Van Gogh]]
- Fauves
