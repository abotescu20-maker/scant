# stone relief (sunk / raised)

## Technique

Figure carved from a stone block so that it projects from (raised) or is cut into (sunk) the background plane. Egyptian preference for sunk relief on sunny exterior walls exploits the hard shadow thrown by the recessed edge; Greek and Roman raised relief creates fuller illusion of bodies emerging from the wall.
