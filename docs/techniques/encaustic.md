# encaustic

## Technique

Pigments suspended in hot beeswax applied to a wood panel with heated spatulas. Produces extraordinarily durable, slightly glossy surface with a buttery impasto; can model flesh with visible tool strokes. Used for Fayum mummy portraits with startling lifelike effect.
