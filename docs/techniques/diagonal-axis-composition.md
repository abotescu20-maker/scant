# Diagonal axis composition

## Technique

A layout principle in which the dominant compositional axis tilts at roughly 30-45° from the horizontal, replacing the classical horizontal/vertical "architectural" grid. Rectangles, photographs, and blocks of type are rotated together onto a single slanted line of movement; negative space is consciously asymmetrical.

Characteristic features:

- Rotated headlines — type running upward into the corner
- Photomontaged figure cropped along a diagonal edge rather than a rectangle
- Wedges, arrows, and bars pointing into the composition
- Strong tension between the tilted axis and the physical rectangle of the page
- Reads as dynamic, propulsive, "revolutionary" — the opposite of classical symmetry
