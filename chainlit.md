# Alex — Asistentul AI pentru Brokeri de Asigurari

**Colegul tau digital — disponibil 24/7, in romana, engleza si germana.**

---

## Ce pot face pentru tine:

### Clienti (5 operatiuni)
- **Cauta client** — dupa nume, telefon, email sau ID
- **Afiseaza profil complet** — polite active, daune, oferte, scor compliance
- **Creeaza client nou** — persoana fizica sau juridica, RO/DE
- **Actualizeaza date** — email, telefon, adresa, note
- **Sterge client** — cu toate datele asociate

### Produse si Oferte (5 operatiuni)
- **Cauta produse** — RCA, CASCO, PAD, CMR, KFZ, HEALTH, LIFE, LIABILITY, GEBÄUDE
- **Compara produse** — tabel comparativ side-by-side
- **Genereaza oferta** — text profesional + PDF/XLSX/DOCX descarcabil
- **Listeaza oferte** — istoricul ofertelor per client
- **Trimite oferta pe email** — direct din chat catre adresa clientului

### Polite si Reinnoiri (2 operatiuni)
- **Dashboard reinnoiri** — ce expira in 7 / 30 / 45 zile, filtrat pe tip
- **Listeaza polite** — portofoliu complet al unui client sau al firmei

### Daune (2 operatiuni)
- **Inregistreaza dauna** — dosar nou cu toate detaliile
- **Statusul daunei** — urmarire dosar deschis sau inchis

### Rapoarte si Compliance (5 operatiuni)
- **Raport ASF** — raport lunar conform Legii 236/2018 (Romania)
- **Raport BaFin** — raport lunar conform VVG (Germania)
- **Valabilitate RCA** — verifica din baza de date a firmei
- **Cross-sell** — ce produse ii lipsesc clientului? bundle recomandat
- **Verificare dosar** — scor compliance 0-100, documente lipsa, riscuri

### Calculator si Estimari (1 operatiune)
- **Calculator prima** — estimare RCA/CASCO pe baza datelor vehiculului

### Documente si Cunostinte (6 operatiuni)
- **Upload document** — polita scanata, CI, constatare amiabila, factura
- **Analizeaza document** — Alex extrage datele relevante automat
- **Cauta in documente** — cautare semantica in documentele incarcate
- **Status baza cunostinte** — cate documente indexate
- **Reindexa documente** — actualizeaza baza de cautare

### Cloud Storage (6 operatiuni)
- **Google Drive** — incarca, listeaza, obtine link descarcare
- **SharePoint** — incarca, listeaza, obtine link descarcare

### Automatizare Desktop (3 operatiuni)
- **Status agent local** — verifica daca agentul local este conectat
- **Ruleaza task** — trimite instructiune catre agentul local (browser/desktop)
- **Navigare web** — deschide URL si extrage informatii din pagini web

### Conversatii (1 operatiune)
- **Salveaza conversatia** — leaga discutia de un client pentru acces ulterior
